# Report temporanei del progetto SmartBack

Questa cartella raccoglie il registro di sviluppo del progetto durante la fase di realizzazione.

## Scopo

- tenere traccia delle modifiche effettuate;
- annotare verifiche, decisioni e problemi rilevanti;
- facilitare la preparazione del resoconto e della documentazione finale;
- conservare il contesto necessario per le attività successive.

## Struttura

- `REGISTRO_MODIFICHE.md`: registro cronologico unico, aggiornato dopo ogni intervento concreto.

Ogni voce indica data, obiettivo, file o componenti interessati, modifiche, verifiche e note ancora aperte.

## Natura temporanea

Questa cartella non fa parte del prodotto finale. Al termine del progetto dovrà essere eliminata, dopo aver trasferito le informazioni utili nella documentazione definitiva.
